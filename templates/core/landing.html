About This Project

Mila is a real-time chat application built as a university project (SE-23-03) at the University of Sargodha, supervised by Dr. Ahmad Mustafa.

Team: Muhammad Abu Bakar, Tehzeeb Shahid, Mir Mohammad

Built with Django, WebSockets, and modern web technologies, Mila demonstrates a complete real-time communication platform with professional UI/UX design.
